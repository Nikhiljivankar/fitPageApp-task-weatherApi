firstly install redis server on you system and run it 


then install packages using following command
npm i 

create .env file in root directory
and add env variables values 
see example.env  file

then to run project use following command
node index.js or 
npm run start or
npm run dev

to see api documentation please  visit http://localhost:3000/api/v1/docs
if you have use another port number replace it with 3000 

added postman collection for api reference
